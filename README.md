## 專案說明
切版模版

## 開始使用

1. Clone Repository

```
git clone https://github.com/mirageapache/template.git
```

2.安裝 npm

```
npm install
```

3.啟動程式

```
npx webpack serve
```

4.瀏覽網頁

```
瀏覽 http://localhost:8080/
```

5.終止程式

```
在終端機按下兩次 ctrl + C 即可終止程式
```



## 環境設置

- node - 18.20.4
- webpack - 5.95.0
